# 行動規範 (Code of Conduct)

## 🎯 私たちの約束

日本株ウォッチドッグコミュニティの一員として、私たちはハラスメントのない環境を作り、年齢、体型、目に見える・見えない障害、民族性、性的指向、性別、性別表現、性同一性、経験レベル、教育、社会経済的地位、国籍、外見、人種、宗教、または政治的信念に関係なく、すべての人が参加できるようにすることを約束します。

私たちは、オープンで歓迎的で、多様で、包括的で、健全なコミュニティに貢献するような方法で行動し、相互作用することを約束します。

## 📋 私たちの基準

### ✅ 肯定的な環境に貢献する行動の例：

- **他者への共感と親切さを示す**
- **異なる意見、観点、経験を尊重する**
- **建設的フィードバックを礼儀正しく与え、優雅に受け入れる**
- **間違いを認め、それによって影響を受けた人々に謝罪し、経験から学ぶ**
- **個人だけでなく、コミュニティ全体にとって最良のことに焦点を当てる**
- **投資に関する多様なアプローチと戦略を尊重する**
- **初心者の質問を歓迎し、学習を支援する**

### ❌ 受け入れられない行動の例：

- **性的な言葉や画像の使用、および性的な注意や進歩**
- **トローリング、侮辱的・軽蔑的コメント、および個人的・政治的攻撃**
- **公的または私的ハラスメント**
- **明示的な許可なしに他者の個人情報（物理的または電子メールアドレスなど）を公開する**
- **投資パフォーマンスや財務状況に関する嘲笑や差別**
- **投資助言や保証を装った詐欺的行為**
- **専門的環境で合理的に不適切と考えられるその他の行為**

## 🔒 投資コミュニティ特有のガイドライン

### 財務情報の取り扱い
- **個人の投資成績**: 他者の損益に対する批判や嘲笑は禁止
- **投資アドバイス**: 個人的な意見は明確にマークし、投資助言として提示しない
- **リスク開示**: 投資にはリスクが伴うことを常に念頭に置く
- **詐欺防止**: 怪しい投資勧誘や詐欺的な情報共有を拒否

### 技術的議論
- **コード品質**: 建設的な技術レビューを促進
- **セキュリティ**: 財務データの安全性を最優先
- **プライバシー**: 個人の投資データの保護を重視

## 🛡️ 執行責任

コミュニティのリーダーは、この行動規範を明確にし、一貫して公正に執行する責任があり、不適切、脅迫的、攻撃的、または有害と判断されるコメント、コミット、コード、wiki編集、問題、およびその他の貢献に対応して適切な是正措置を講じます。

コミュニティのリーダーは、この行動規範に沿わないコメント、コミット、コード、wiki編集、問題、およびその他の貢献を削除、編集、または拒否する権利と責任を有し、適切な場合にはモデレーション決定の理由を伝えます。

## 📍 適用範囲

この行動規範は、すべてのコミュニティスペース内で適用され、個人が公的スペースでコミュニティを代表している場合にも適用されます。私たちのコミュニティを代表する例には以下が含まれます：

- **GitHub**: Issues, Pull Requests, Discussions
- **公式メールアドレス**: プロジェクト関連の連絡
- **ソーシャルメディア**: プロジェクトの公式アカウント
- **イベント**: オンライン・オフラインのイベントでの代表

## ⚖️ 執行

### 報告方法
嫌がらせ、虐待、またはその他の受け入れられない行動の事例は、執行を担当するコミュニティリーダーに以下の方法で報告できます：

- **GitHub Issues**: セキュリティや行動規範に関する報告
- **メール**: conduct@example.com
- **プライベートメッセージ**: GitHub経由で管理者に直接連絡

すべての苦情は迅速かつ公正にレビューおよび調査されます。

### 執行プロセス
1. **初期対応**: 24時間以内に受領確認
2. **調査**: 関係者からの情報収集
3. **決定**: コミュニティリーダーによる判断
4. **措置**: 適切な是正措置の実施
5. **フォローアップ**: 必要に応じて継続的な監視

## 📏 執行ガイドライン

コミュニティリーダーは、この行動規範に違反したと判断された行動に対する結果を決定する際に、以下のコミュニティ影響ガイドラインに従います：

### 1. 是正措置
**コミュニティへの影響**: 不適切な言葉の使用またはその他の職業的でない、またはコミュニティで歓迎されないと判断された行動。

**結果**: コミュニティリーダーからの私的な書面による警告、違反の性質の明確化、および行動が不適切だった理由の説明。公開謝罪が要求される場合があります。

### 2. 警告
**コミュニティへの影響**: 単一の事件または一連の行動による違反。

**結果**: 継続的な行動に対する結果を伴う警告。指定された期間、行動規範を適用する人々との一方的な相互作用や公的コミュニケーションを含む、関係者との相互作用なし。これには、ソーシャルメディアなどの外部チャネルでのコミュニティスペースや一方的な相互作用の回避が含まれます。これらの条件に違反すると、一時的または永続的な禁止につながる可能性があります。

### 3. 一時的な禁止
**コミュニティへの影響**: 持続的な不適切な行動を含む、コミュニティ基準の重大な違反。

**結果**: 指定された期間、コミュニティとのあらゆる種類の相互作用や公的コミュニケーションの一時的な禁止。この期間中、行動規範を適用する人々との公的または私的な相互作用は許可されません。これらの条件に違反すると、永続的な禁止につながる可能性があります。

### 4. 永続的な禁止
**コミュニティへの影響**: 持続的な不適切な行動、個人の嫌がらせ、または個人のクラスに対する攻撃や軽蔑を含む、コミュニティ基準の違反のパターンを示すこと。

**結果**: コミュニティ内でのあらゆる種類の公的相互作用の永続的な禁止。

## 🤝 紛争解決

### ステップ1: 直接解決
可能な場合は、まず関係者間で問題を直接解決することを試みてください。

### ステップ2: 仲裁
直接解決が困難な場合は、中立的な第三者による仲裁を求めることができます。

### ステップ3: 公式報告
上記のステップが失敗した場合、または重大な違反の場合は、公式の報告プロセスを使用してください。

## 📞 緊急時の連絡

### 即座の対応が必要な場合
- **セキュリティ問題**: security@example.com
- **深刻な嫌がらせ**: 法執行機関への連絡を検討
- **技術的緊急事態**: GitHub Issues（緊急ラベル付き）

## ❓ よくある質問

**Q: この行動規範はいつ適用されますか？**
A: プロジェクトに関連するすべての活動において適用されます。

**Q: 匿名で報告できますか？**
A: はい、ただし詳細な調査には限界がある場合があります。

**Q: 投資助言に関する制限はありますか？**
A: 個人的な意見として明確にマークする限り、投資に関する議論は歓迎されます。

**Q: 違反報告後、何が起こりますか？**
A: 24時間以内に受領確認を送信し、調査を開始します。
